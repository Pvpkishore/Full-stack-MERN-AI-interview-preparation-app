# Full-stack MERN AI Interview Preparation App

This repository contains the code for a Full-stack MERN application designed for AI-based interview preparation.

## Features
- Comprehensive set of questions and answers.
- AI-based ranking and suggestion system.
- User authentication and management.

## Installation
1. Clone the repo: `git clone https://github.com/Pvpkishore/Full-stack-MERN-AI-interview-preparation-app.git`
2. Install dependencies for the server and client.
3. Start the server and client applications.
